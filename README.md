# VibeThinker-3B Claim Check

This project is a small independent evaluation of VibeThinker-3B on verifiable reasoning tasks.

The goal is not to reproduce the full paper, but to test some of its claims on a local 8GB GPU setup using math and coding problems.

## Goals

- Run VibeThinker-3B locally
- Test it on small math and coding tasks
- Track accuracy, latency, and failure cases
- Compare results against another small baseline model
